IS JavaScript Even an OOP Language?

The question of whether JavaScript is an OOP is active. Some think YES and some NO! Mostly, it is considered an OOP language, and, from the point of all truth, Wilipeia lists it as an OOP.

What Makes JavaScript an OOP?

Javascript meets all three of these tests to determine that it is an OOP

Association: an object’s ability to refer to another object
Aggregation: an object’s capability to include more objects
Composition: an object can embed other objects
